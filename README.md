# WorkFlow

WorkFlow is a focused task flow app that keeps daily work clear, fast, and easy to scan. Built with Svelte, TypeScript, and Vite for a snappy UI and clean iteration.

## Features

- Quick task capture with a minimal, distraction-free layout.
- Status-driven workflow: plan, do, review.
- Compact list views that surface what matters right now.
- Keyboard-first interactions for speed.
- Local state that keeps the UI responsive.

## Planned

- Saved projects with per-project task lists.
- Tags and filters for sharper focus.
- Drag-and-drop ordering.
- Export and backup options.
- Theme presets and layout density controls.

## Getting started

```bash
npm install
npm run dev
```
